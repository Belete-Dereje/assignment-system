# Distributed Assignment Submission & Evaluation System

A full-featured distributed web application for managing assignment submissions and evaluations across multiple machines.

## Architecture
- Ubuntu + Windows nodes via Zerotier
- Nginx load balancer
- SQLite database per node
- HTTP Sync every 5 seconds
- ngrok public access

## Quick Start
python app.py
Default Admin: ADMIN001 / admin@system.com / admin123
